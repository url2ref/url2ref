## Translation

Enable in Options panel. Requires `DEEPL_API_KEY` or `GOOGLE_TRANSLATE_API_KEY`. Output includes both original and translated title.

## Zotero/Citoid

Enable in Options panel. Uses Wikipedia's Citoid API with specialized extractors for academic databases and complex pages.

## AI Extraction

Enable in Options panel. Enter API key (OpenAI or Anthropic). Key is used client-side and not stored on server.

## Archive Integration

Automatically checks Wayback Machine for existing archives. Shows archive URL/date or offers to create one.
