# Web Interface

Browser-based citation generation at `http://localhost:8000`.

## Start

```bash
cargo run --bin url2ref-web
```

## Features

- Multiple formats (Wiki, BibTeX, Harvard)
- Compare metadata from all sources
- One-click copy to clipboard
- Title translation (DeepL/Google)
- AI extraction (OpenAI/Anthropic)
- Wayback Machine archive integration
